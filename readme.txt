Unlimited-level cascading mega menu with full Elementor widget support.

== Description ==

The **Cascading Mega Menu for Elementor** plugin allows you to create a dynamic, unlimited-level cascading mega menu directly within the Elementor editor.

**Key Features:**
*   **Unlimited Nesting:** Create menus with as many levels as you need.
*   **Elementor Native:** Built as a standard Elementor widget for easy drag-and-drop usage.
*   **Responsive Design:** Works seamlessly on desktops and mobile devices.
*   **Customizable:** Control colors, fonts, spacing, and icons to match your site's design.
*   **Dynamic Headings:** Option to show parent item titles as headings in submenus.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/cascading-mega-menu` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Open the Elementor editor and drag the "Cascading Mega Menu" widget to your desired location.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =
No, this works with the free version of Elementor.

= Can I use custom icons? =
Yes, the widget supports Elementor's icon library.
